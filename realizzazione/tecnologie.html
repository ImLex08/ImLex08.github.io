Tecnologie Utilizzate

Frontend

  • HTML5 - Per la struttura base del sito
  • CSS3 - Per lo stile e il responsive design
  • JavaScript ES6+ - Per l'interattività
  • Web Components - Per la modularità

Strumenti

  • Git & GitHub - Per il version control e hosting
  • Visual Studio Code - Come editor principale
  • GitHub Pages - Per il deployment automatico

Processo di Sviluppo

Fasi Principali

  1. Progettazione: Definizione struttura e contenuti

    Scelta di un'architettura modulare basata su componenti riutilizzabili

  2. Sviluppo: Implementazione incrementale

    Creazione prima dei componenti base, poi delle singole pagine

  3. Testing: Verifica cross-browser

    Test su Chrome, Firefox e mobile devices

  4. Deploy: Pubblicazione automatica

    Configurazione GitHub Actions per il deployment su GitHub Pages

Scelta di Hosting

GitHub Pages

Scelto per:

  • Integrazione nativa con repository GitHub
  • Deployment automatico ad ogni push
  • SSL gratuito
  • Performance affidabili

Configurazione tramite file render.yaml nella root del progetto
